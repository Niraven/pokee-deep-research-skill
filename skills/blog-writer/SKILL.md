# Blog Writer

Write blog posts with SEO optimization and proper structure.

## Output Format

```markdown
# Title

## Introduction
Hook + problem statement

## Main Points
- H2 sections
- Bullet lists
- Code examples

## Conclusion
Summary + CTA
```

## Best Practices

- Start with outline
- Use headers for structure
- Include code examples
- End with clear CTA
