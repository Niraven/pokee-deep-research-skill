# Gog (Google Workspace)

Google Workspace CLI for Gmail, Calendar, Drive, Contacts, Sheets, and Docs.

## Services

- Gmail: Read, send, manage emails
- Calendar: Events, availability
- Drive: File management
- Sheets: Data operations
- Docs: Document editing

## Auth

Uses OAuth tokens stored in keyring.
Token format: `token:default:email@example.com`
